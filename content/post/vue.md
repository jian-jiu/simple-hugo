## 开始创建项目
项目名称 可以不用
项目产品名称（如果构建移动应用程序，必须以字母开头）（Quasar应用程序）

项目名、作者名之类
选择 css 预处理器
是否使用 eslint：eslint 为 JS 语法校验， 会帮你检查 js 语法错误和形成团队的代码规范。
是否支持 ie11
是否自动安装依赖包，以及哪种形式安装（yarn 或 npm)： 没装 yarn 的话还是选否吧，npm 安装包真的很慢...
